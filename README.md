# NextRole

AI-Powered Job Application Assistant

## About

NextRole uses AI to analyze your resume against job descriptions, giving you instant insights on match scores and skill gaps.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
